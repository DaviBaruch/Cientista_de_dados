## 🚀 Sobre o Projeto

Este projeto foi desenvolvido durante o curso de **Ciência de Dados da Alura**, com o objetivo de aplicar na prática conceitos fundamentais como:

- 📊 Análise Exploratória de Dados (EDA)
- 🧹 Tratamento e limpeza de dados
- 📈 Visualização de dados
- 🤖 Introdução a Machine Learning

Projetos de Data Science são fundamentais para transformar dados em decisões estratégicas, sendo muito utilizados no mercado atual. :contentReference[oaicite:1]{index=1}

---

## 🧠 Objetivo

O principal objetivo deste projeto é:

✔️ Extrair insights relevantes a partir dos dados  
✔️ Identificar padrões e tendências  
✔️ Criar visualizações claras e informativas  
✔️ Aplicar técnicas de análise de dados na prática
